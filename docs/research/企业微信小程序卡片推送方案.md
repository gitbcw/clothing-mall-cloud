# 企业微信小程序卡片推送方案

## 背景与目标

**目标**：在企业微信中，运营人员可以向客户批量发送小程序卡片，客户点击卡片即可直接跳转到小程序指定页面（商品详情、活动页等），实现"触达路径最小化"。

**核心区别**：
| 形式 | 交互方式 | 跳转路径 |
|------|----------|----------|
| 小程序卡片 | 点击直接跳转 | 1步直达 |
| 海报+二维码 | 扫码识别 | 3步（识别→扫码→跳转）|

## API 详解

### 1. 获取 access_token

```
GET https://qyapi.weixin.qq.com/cgi-bin/gettoken?corpid=CORPID&corpsecret=SECRET
```

**响应**：
```json
{
  "errcode": 0,
  "errmsg": "ok",
  "access_token": "xxxxx",
  "expires_in": 7200
}
```

> 已有实现：`WeWorkService.getAccessToken()` 带缓存

### 2. 上传封面图到素材库

```
POST https://qyapi.weixin.qq.com/cgi-bin/media/upload?access_token=ACCESS_TOKEN&type=image
```

**请求**：multipart/form-data，图片文件

**响应**：
```json
{
  "errcode": 0,
  "errmsg": "ok",
  "type": "image",
  "media_id": "xxxxx",
  "created_at": 1234567890
}
```

> 注意：media_id 有效期 3 天，需要定期刷新或使用永久素材

### 3. 发送小程序卡片

```
POST https://qyapi.weixin.qq.com/cgi-bin/externalcontact/add_msg_template?access_token=ACCESS_TOKEN
```

**请求体**：
```json
{
  "chat_type": "single",
  "external_userid": ["外部联系人ID1", "外部联系人ID2"],
  "sender": "企业员工ID",
  "attachments": [
    {
      "msgtype": "miniprogram",
      "miniprogram": {
        "appid": "小程序AppID",
        "title": "商品名称",
        "pic_media_id": "封面图media_id",
        "page": "/pages/goods/goods?id=123"
      }
    }
  ]
}
```

**参数说明**：

| 参数 | 必填 | 说明 |
|------|------|------|
| chat_type | 是 | `single` 单聊，`group` 群聊 |
| external_userid | 是* | 外部联系人ID列表（与 filter 二选一） |
| filter | 是* | 按标签筛选客户（与 external_userid 二选一） |
| sender | 是 | 发送者企业员工ID |
| appid | 是 | 小程序AppID（必须已关联企业微信） |
| title | 是 | 小程序卡片标题，最长64字节 |
| pic_media_id | 是 | 封面图 media_id |
| page | 否 | 小程序页面路径，默认首页 |

**按标签群发示例**：
```json
{
  "chat_type": "single",
  "sender": "员工ID",
  "filter": {
    "tag_list": ["标签ID"]
  },
  "attachments": [
    {
      "msgtype": "miniprogram",
      "miniprogram": {
        "appid": "wx1234567890",
        "title": "春季新品上市",
        "pic_media_id": "media_xxxxx",
        "page": "/pages/activity/spring"
      }
    }
  ]
}
```

## 现有代码复用

### 可复用组件

| 组件 | 文件 | 复用内容 |
|------|------|----------|
| Token 管理 | `WeWorkService.java` | access_token 获取与缓存 |
| HTTP 调用 | `WeWorkService.java` | `postMessageTemplate()` 方法 |
| 系统配置 | `SystemConfig.java` | corpId、secret 等配置项 |

### 需要新增

| 功能 | 说明 |
|------|------|
| 上传素材 | 新增 `uploadMedia()` 方法上传封面图 |
| 发送小程序卡片 | 新增 `sendMiniProgramCard()` 方法 |
| 封面图生成 | 复用 `QCodeService` 裁剪商品图为 5:4 比例 |
| 管理后台 UI | 商品选择、客户/标签选择、批量发送界面 |
| 客户映射 | 用户 external_userid 与本地用户的关联 |

## 前置准备（运营侧）

1. **关联小程序**
   - 企业微信管理后台 → 应用管理 → 小程序 → 关联
   - 需要小程序管理员确认

2. **获取配置信息**
   - 企业 ID (corpId)
   - 客户联系 Secret
   - 发送者员工 ID

3. **准备素材**
   - 设计卡片封面模板
   - 建议尺寸：500x400px (5:4 比例)

## 注意事项

1. **发送限制**
   - 每个客户每天最多接收 1 条企业群发消息
   - 单次群发最多 20000 个客户

2. **权限要求**
   - 发送者需要有「客户联系」应用权限
   - 小程序必须已关联企业微信

3. **media_id 有效期**
   - 临时素材：3 天
   - 永久素材：长期有效（推荐）
   - 建议：实现自动刷新机制

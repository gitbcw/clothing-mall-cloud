## 六、关键结论

1. **时间投入**：10-18 人天，全职约 2-3 周
2. **人力成本**：5,000-9,000 元（按 500 元/天计）
3. **首年支出**：~700 元（不含人力）
4. **后续年度支出**：~1,200-1,700 元/年
5. **成本大头**：人力成本占 80% 以上
6. **客户承担**：微信支付手续费 0.6%，不走我们账目

## 七、同城配送对接成本分析

### 7.1 各平台技术可行性

| 平台 | 开放平台地址 | API 调用费 | 接入门槛 | 技术可行性 |
|------|-------------|-----------|----------|-----------|
| **美团跑腿** | [配送技术服务中心](https://peisong.meituan.com/tscc/) | 免费 | ⚠️ 高：需二级等保 + 安全产品 + 保证金 5-20 万 | 中小商家建议走第三方聚合 |
| **顺丰同城** | [顺丰开放平台](https://open.sf-express.com) | 免费 | 低：注册账号 + 申请接入 | ✅ 可直接对接 |
| **蜂鸟即配** | [蜂鸟开放平台](https://fengniao.ele.me/business.html) | 免费 | 中：需商家账号 + OAuth 授权 | ✅ 可直接对接 |
| **达达快送** | [新达达开放平台](https://newopen.imdada.cn/) | 免费 | 低：开发者 + 商户账号 | ✅ 可直接对接 |

### 7.2 配送费用（顾客/商家承担）

| 平台 | 配送费计算方式 | 费用范围 |
|------|---------------|---------|
| 美团跑腿 | 3km 起步 + 距离 + 金额 | 起步 ~3.15 元，超 3km 每 0.1km +0.2 元 |
| 顺丰同城 | 按距离/时段 | 约 10-30 元/单 |
| 蜂鸟即配 | 按距离/时段 | 约 8-25 元/单 |
| 达达快送 | 按距离/时段 | 约 8-20 元/单 |

> **说明**：配送费由顾客下单时预付，不经过我们系统结算。

### 7.3 第三方聚合平台（推荐方案）

如果直接对接美团门槛太高，可考虑使用聚合配送平台：

| 平台 | 说明 | 费用 |
|------|------|------|
| 快递100 同城配送 | 聚合多家配送公司 | [官网](https://api.kuaidi100.com/product/local/) |
| 快鸟数据 | 聚合物流 API | 按调用次数收费 |

**优势**：
- 一次接入，多家运力
- 门槛低，无需各平台单独审核
- 可比价选择最优运力

### 7.4 开发成本估算

| 项目 | 工作量 | 说明 |
|------|--------|------|
| 单平台对接 | 2-3 天 | 选定一个平台，完成 API 对接 |
| 聚合平台对接 | 3-4 天 | 对接聚合平台，获得多运力 |
| 多平台分别对接 | 5-8 天 | 分别对接 2-3 个平台 |

### 7.5 运营成本

| 项目 | 费用 | 说明 |
|------|------|------|
| API 调用 | 0 元 | 各平台 API 调用本身免费 |
| 配送费 | 8-30 元/单 | **顾客承担**，系统仅做费用计算和预收 |
| 开发对接 | 2-8 天 | 根据方案选择 |

### 7.6 同城配送对接结论

| 结论 | 说明 |
|------|------|
| **技术上完全可行** | 四大平台均有开放 API，顺丰/达达/蜂鸟门槛较低 |
| **API 调用无额外费用** | 各平台 API 调用免费，不增加运营成本 |
| **推荐方案** | MVP 阶段先对接 1-2 个低门槛平台（顺丰同城/达达），后续可扩展聚合平台 |
| **成本影响** | 对我们几无影响，配送费由顾客承担 |

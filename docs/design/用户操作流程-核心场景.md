# 用户操作流程 · 核心场景

> **用途**：给客户讲解用户如何使用系统
> **版本**：v1.0
> **更新时间**：2026-03-04

## 场景一：浏览商品并下单

### 从首页轮播图进入

```
首页轮播图
    │
    ↓ 点击
商品详情页
    │
    ├── 查看商品图片（左右滑动）
    ├── 查看价格、原价、销量
    ├── 查看商品描述
    ├── 选择颜色、尺码 → 显示库存情况
    │
    ↓ 点击「立即购买」
订单确认页
    │
    ├── 选择/新增收货地址
    ├── 选择配送方式
    │   ├── 快递发货 → 显示运费
    │   ├── 到店自提 → 显示自提点、自提码
    │   └── 同城配送 → 显示配送费、预计送达时间
    ├── 选择优惠券（如有）
    ├── 填写备注（可选）
    ├── 查看实付金额
    │
    ↓ 点击「提交订单」
支付页面
    │
    ↓ 微信支付
支付结果
    │
    ├── 成功 → 可查看订单 / 继续购物
    └── 失败 → 重新支付 / 取消订单
```

### 从分类导航进入

```
首页分类图标（如"上装"）
    │
    ↓ 点击
分类商品列表
    │
    ├── 按风格/季节筛选（可选）
    ├── 按销量/价格排序
    ├── 浏览商品卡片
    │
    ↓ 点击商品
商品详情页
    │
    └── （后续同上）
```

### 从搜索进入

```
首页搜索框
    │
    ↓ 点击，输入关键词
搜索结果列表
    │
    ├── 点击历史搜索词（快速搜索）
    ├── 或点击热门搜索词
    ├── 或手动输入后搜索
    │
    ↓ 点击商品
商品详情页
    │
    └── （后续同上）
```

## 场景二：参与限时折扣活动

### 活动形式说明

| 形式 | 说明 | 用户感知 |
|------|------|----------|
| 直接打折 | 商品显示原价+折扣价 | 结算时自动按折扣价，无需额外操作 |
| 满减活动 | 满 X 元减 Y 元 | 结算时自动抵扣，显示"已减 Y 元" |
| 领券折扣 | 需先领券再使用 | 领券后，结算时选择使用该券 |

### 直接打折流程

```
首页「限时折扣」入口
    │
    ↓ 点击
折扣专区
    │
    ├── 查看折扣商品列表
    ├── 每个商品显示：原价（划线）+ 折扣价
    ├── 活动倒计时
    │
    ↓ 点击商品
商品详情页
    │
    ├── 价格显示折扣价
    ├── 选择规格
    │
    ↓ 立即购买
订单确认页
    │
    ├── 商品金额显示折扣价
    ├── 无需额外操作
    │
    ↓ 提交订单并支付
```

### 满减活动流程

```
首页「限时折扣」入口
    │
    ↓ 点击
折扣专区
    │
    ├── 查看活动规则（满 200 减 20）
    ├── 折扣商品列表
    │
    ↓ 选择商品，加入购物车
购物车
    │
    ├── 选择多个商品
    ├── 查看合计金额
    │
    ↓ 结算
订单确认页
    │
    ├── 系统自动判断是否满足满减条件
    ├── 满足：显示「满减优惠 -20 元」
    ├── 不满足：提示「再买 X 元可减 Y 元」
    │
    ↓ 提交订单并支付
```

### 领券后折扣流程

```
首页「限时折扣」入口 或 「新人专享」
    │
    ↓ 点击
活动页面
    │
    ├── 看到「领券」按钮
    │
    ↓ 点击领券
领券成功
    │
    ├── 提示「优惠券已发放到您的账户」
    │
    ↓ 去选购商品
商品详情页 → 订单确认页
    │
    ├── 在「优惠券」位置看到可用优惠券
    │
    ↓ 点击选择优惠券
    │
    ├── 显示优惠金额
    ├── 更新实付金额
    │
    ↓ 提交订单并支付
```

## 场景三：查看订单与确认收货

### 查看订单列表

```
个人中心
    │
    ↓ 点击「全部订单」
订单列表
    │
    ├── Tab 筛选：全部 / 待支付 / 待发货 / 待自提 / 待收货 / 已完成
    ├── 每个订单卡片显示：
    │   ├── 商品图片、名称、规格
    │   ├── 订单状态
    │   ├── 订单金额
    │   └── 操作按钮
    │
    ↓ 点击订单
订单详情
    │
    ├── 订单信息（编号、下单时间、状态）
    ├── 商品信息
    ├── 价格明细
    ├── 物流信息
    └── 操作按钮
```

### 快递发货 - 确认收货

```
订单详情
    │
    ├── 配送方式：快递发货
    ├── 物流信息：物流公司 + 单号 + 运输节点
    │
    ↓ 收到货后，点击「确认收货」
确认弹窗
    │
    ↓ 确认
收货成功
    │
    └── 订单状态变为「已完成」
```

### 到店自提 - 核销取货

```
订单详情
    │
    ├── 配送方式：到店自提
    ├── 显示自提码（二维码/数字码）
    ├── 显示自提点地址、营业时间
    │
    ↓ 顾客到店，出示自提码
店员扫码核销
    │
    ↓ 核销成功
订单完成
    │
    └── 订单状态变为「已完成」
```

### 同城配送 - 签收

```
订单详情
    │
    ├── 配送方式：同城配送
    ├── 骑手信息（姓名、电话）
    ├── 实时位置/预计送达
    │
    ↓ 骑手送达
顾客签收
    │
    └── 订单状态自动变为「已完成」
```

## 场景四：申请换货（只换不退模式）

### 发起换货申请

```
订单详情（已完成的订单）
    │
    ↓ 点击「申请售后」
选择售后商品
    │
    ├── 勾选要换的商品
    │
    ↓ 下一步
选择售后类型
    │
    ├── 换货（当前店铺仅支持换货）
    │
    ↓ 选择
选择换货规格
    │
    ├── 选择新颜色
    ├── 选择新尺码
    ├── 系统自动计算差价
    │   ├── 无差价：无需补款
    │   ├── 需补差价：显示补差金额
    │   └── 可退差价：显示退差金额
    │
    ↓ 确认
填写申请信息
    │
    ├── 选择换货原因（尺码不合适/颜色差异/质量问题）
    ├── 问题描述（选填）
    ├── 上传图片（选填）
    │
    ↓ 提交申请
申请提交成功
    │
    └── 等待商家审核
```

### 商家审核通过后

```
售后详情
    │
    ├── 审核状态：已通过
    │
    ├── 换货方式
    │   │
    │   ├── 方式一：到店换货
    │   │   ├── 显示核销码
    │   │   ├── 携带原商品到店
    │   │   ├── 店员核销后换取新商品
    │   │   └── 如需补差价，现场支付
    │   │
    │   └── 方式二：寄回换货
    │       ├── 填写物流单号
    │       ├── 寄回原商品
    │       ├── 商家验货
    │       ├── 如需补差价，在线支付
    │       └── 商家发出新商品
    │
    ↓ 换货完成
订单状态更新
```

## 场景七：企业管理员操作（运营后台）

### 7.1 新增商品

```
运营后台
    │
    ↓ 左侧菜单「商品管理」→「商品列表」
商品列表页
    │
    ↓ 点击「新增商品」
新增商品表单
    │
    ├── 填写基础信息
    │   ├── 商品名称
    │   ├── 商品分类
    │   ├── 商品描述
    │   ├── 上传主图（支持多张）
    │   └── 上传详情图
    │
    ├── 设置价格库存
    │   ├── 销售价
    │   ├── 原价（划线价）
    │   └── 总库存
    │
    ├── 设置 SKU 规格
    │   ├── 添加颜色规格（如：黑色、白色、蓝色）
    │   ├── 添加尺码规格（如：S、M、L、XL）
    │   ├── 为每个 SKU 设置价格和库存
    │   └── 为每个 SKU 上传图片
    │
    ├── 其他设置
    │   ├── 商品标签
    │   └── 关联供应商（可选）
    │
    ↓ 点击「保存并上架」
商品上架成功
    │
    └── 小程序端可看到该商品
```

### 7.2 订单发货

```
运营后台
    │
    ↓ 左侧菜单「订单管理」→「订单列表」
订单列表
    │
    ├── 筛选「待发货」订单
    │
    ↓ 点击订单「发货」按钮
发货弹窗
    │
    ├── 快递发货
    │   ├── 选择物流公司
    │   ├── 输入物流单号
    │   └── 确认发货
    │
    ├── 到店自提（备货完成）
    │   └── 确认备货 → 通知顾客到店
    │
    └── 同城配送
        ├── 选择配送平台
        ├── 确认配送费
        └── 发起配送
    │
    ↓ 确认
发货成功
    │
    ├── 订单状态更新
    └── 顾客收到发货通知
```

### 7.3 审核换货申请

```
运营后台
    │
    ↓ 左侧菜单「售后管理」→「售后列表」
售后列表
    │
    ├── 筛选「待审核」
    │
    ↓ 点击订单
售后详情
    │
    ├── 查看申请信息
    │   ├── 换货商品
    │   ├── 换货规格（新颜色/新尺码）
    │   ├── 差价情况
    │   └── 换货原因
    │
    ↓ 点击「通过」
确认换货
    │
    ├── 确认换货商品和规格
    ├── 差价处理
    │   ├── 无差价：直接确认
    │   ├── 需补差价：发送支付链接给顾客
    │   └── 需退差价：确认后系统退款
    │
    ↓ 确认
审核通过
    │
    └── 顾客收到通知，进行换货
```

### 7.4 创建优惠券

```
运营后台
    │
    ↓ 左侧菜单「活动管理」→「优惠券管理」
优惠券列表
    │
    ↓ 点击「新增优惠券」
新增优惠券表单
    │
    ├── 填写基本信息
    │   ├── 优惠券名称（如：新人专享券）
    │   ├── 优惠类型
    │   │   ├── 满减券（满 X 元减 Y 元）
    │   │   ├── 折扣券（打 X 折）
    │   │   └── 无门槛券（直接减 X 元）
    │   ├── 优惠金额/折扣力度
    │   └── 使用门槛（满减券需设置）
    │
    ├── 设置发放规则
    │   ├── 发放总量（如：1000 张）
    │   ├── 每人限领（如：1 张）
    │   └── 适用商品
    │       ├── 全场通用
    │       └── 指定商品（勾选）
    │
    ├── 设置有效期
    │   ├── 固定时间段（如：3 月 1 日 - 3 月 31 日）
    │   └── 或领取后 X 天内有效
    │
    ├── 选择发放方式
    │   ├── 手动发放（后台选择会员发放）
    │   └── 自动发放
    │       ├── 新用户注册自动发
    │       └── 首次下单自动发
    │
    ↓ 点击「保存」
优惠券创建成功
    │
    └── 可在优惠券列表查看、编辑、发放
```

### 7.5 创建限时折扣活动

```
运营后台
    │
    ↓ 左侧菜单「活动管理」→「活动配置」
活动列表
    │
    ↓ 点击「新增活动」→ 选择「限时折扣」
新增限时折扣
    │
    ├── 填写活动信息
    │   ├── 活动名称（如：春季清仓 8 折）
    │   ├── 活动时间（开始时间 - 结束时间）
    │   └── 活动标签（如：限时折扣，会显示在商品上）
    │
    ├── 设置折扣规则
    │   ├── 折扣力度
    │   │   ├── 打折（如：8 折）
    │   │   └── 或直降（如：立减 20 元）
    │   └── 限购数量（如：每人限购 2 件）
    │
    ├── 选择参与商品
    │   ├── 从商品列表勾选
    │   └── 可批量选择
    │
    ↓ 点击「保存并启用」
活动创建成功
    │
    ├── 活动期间，商品显示折扣价
    ├── 小程序首页可配置活动入口
    └── 结算时自动按折扣价计算
```

### 7.6 创建满减活动

```

    │       ├── 门槛金额
    │       └── 减免金额
    │

    │   └── 或指定商品
    │
    ↓ 点击「保存并启用」
活动创建成功

### 7.7 发送消息推送

```
运营后台
    │
    ↓ 左侧菜单「消息推送」
消息推送页面
    │
    ├── 选择推送类型

编辑推送内容

    │   ├── 按等级选择
    │   └── 按条件筛选
    │       ├── 注册时间（如：近 30 天新用户）
    │       ├── 消费金额（如：累计消费 > 500 元）
    │       └── 活跃度（如：30 天内有访问）
    │
    ├── 编辑推送内容
    │   ├── 推送标题
    │   ├── 推送内容
    │   └── 跳转链接
    │       ├── 跳转商品详情
    │       ├── 跳转活动页面
    │       └── 或自定义链接
    │
    ├── 预览推送效果
    │
    ├── 选择发送方式
    │   ├── 立即发送
    │   └── 定时发送（选择日期时间）
    │
    ↓ 点击「发送」
推送成功
    │
    ├── 顾客在企业微信收到消息
    └── 可在「推送记录」查看送达情况

| 用户角色 | 核心场景 | 入口 |
|----------|----------|------|

| 顾客 | 下单购买 | 商品详情页 → 订单确认 → 支付 |
| 顾客 | 参与活动 | 活动入口 → 领券/选商品 → 下单 |
| 顾客 | 查看订单 | 个人中心 → 订单列表 |
| 顾客 | 确认收货 | 订单详情 → 确认收货/核销 |
| 顾客 | 申请换货 | 订单详情 → 申请售后 |
| 顾客 | 收藏/分享 | 商品详情页 |
| 顾客 | 使用优惠券 | 订单确认页 → 选择优惠券 |
| 管理员 | 新增商品 | 商品管理 → 新增商品 |
| 管理员 | 订单发货 | 订单管理 → 发货 |
| 管理员 | 审核售后 | 售后管理 → 审核 |
| 管理员 | 创建优惠券 | 活动管理 → 优惠券管理 |
| 管理员 | 创建限时折扣 | 活动管理 → 活动配置 |
| 管理员 | 创建满减活动 | 活动管理 → 活动配置 |
| 管理员 | 发送消息推送 | 消息推送 |
| 管理员 | 查看数据统计 | 数据统计 |
| 管理员 | 会员管理 | 会员管理 → 会员列表 |
| 管理员 | 管理会员标签 | 会员管理 → 标签管理 |
| 管理员 | 系统设置 | 系统设置 |

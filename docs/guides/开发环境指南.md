## 二、快速开始

### 2.1 一键启动（推荐）

```bash
# 在项目根目录执行
./scripts/docker-start.sh deploy
```

这会自动完成：
1. 初始化配置文件
2. 打包后端 JAR
3. 打包管理前端
4. 启动所有 Docker 服务

### 2.2 常用命令

```bash
./scripts/docker-start.sh start       # 启动服务
./scripts/docker-start.sh stop        # 停止服务
./scripts/docker-start.sh restart     # 重启服务
./scripts/docker-start.sh status      # 查看状态
./scripts/docker-start.sh logs        # 查看所有日志
./scripts/docker-start.sh logs app    # 查看应用日志
./scripts/docker-start.sh --help      # 查看所有命令
```

### 2.3 访问地址

| 服务 | 地址 |
|------|------|
| 管理后台 | http://localhost |
| 后端 API | http://localhost:8088 |
| MySQL | localhost:3306 |

**默认账号**：
- 管理后台：admin123 / admin123
- MySQL：clothing_mall / clothing123456

## 三、环境准备

### 3.1 必需软件

| 软件 | 版本 | 说明 |
|------|------|------|
| Docker Desktop | 最新 | 容器运行环境 |
| JDK | 1.8 | 后端编译（可选，Docker 内置） |
| Node.js | 16.x | 前端编译（可选，Docker 内置） |
| Maven | 3.6+ | 后端构建（可选，Docker 内置） |

### 3.2 国内网络加速

如果镜像拉取失败，可以启用代理模式：

```bash
./scripts/docker-start.sh start --proxy
```

或配置 Docker 镜像加速器（推荐）：

```json
// ~/.docker/daemon.json
{
  "registry-mirrors": [
    "https://mirror.ccs.tencentyun.com",
    "https://docker.mirrors.ustc.edu.cn"
  ]
}
```

## 四、数据库管理

### 4.1 从服务器同步数据库

```bash
./scripts/docker-start.sh sync-db
```

这会从生产服务器导出最新数据库到本地。

### 4.2 手动连接数据库

```bash
# 通过 Docker 连接
docker exec -it clothing-mall-mysql-local mysql -uclothing_mall -pclothing123456 clothing_mall

# 或通过本地客户端
mysql -h 127.0.0.1 -P 3306 -u clothing_mall -pclothing123456 clothing_mall
```

### 4.3 数据库凭据

| 参数 | 值 |
|------|------|
| Host | localhost |
| Port | 3306 |
| Database | clothing_mall |
| Username | clothing_mall |
| Password | clothing123456 |
| Root Password | root123456 |

## 五、开发模式（热重载）

如果需要热重载开发，可以单独启动服务：

### 5.1 仅启动 MySQL

```bash
cd docker && docker compose -f docker-compose.local.yml up mysql -d
```

### 5.2 本地运行后端

```bash
# Maven 构建
mvn clean package -DskipTests -s docker/maven/settings.xml

# 启动
java -Dfile.encoding=UTF-8 -jar clothing-mall-all/target/clothing-mall-all-0.1.0-exec.jar
```

### 5.3 本地运行前端

```bash
cd clothing-mall-admin
npm install --registry=https://registry.npmmirror.com
npm run dev
```

**注意**：Windows 下需设置 `NODE_OPTIONS=--openssl-legacy-provider`

## 七、常见问题

### 7.1 Docker 服务无法启动

```bash
# 查看日志
./scripts/docker-start.sh logs mysql

# 重置环境（危险：删除数据）
./scripts/docker-start.sh clean
./scripts/docker-start.sh deploy
```

### 7.2 前端构建失败

```bash
# Node.js 版本问题
export NODE_OPTIONS=--openssl-legacy-provider
./scripts/docker-start.sh pack-admin
```

### 7.3 数据库初始化失败

```bash
# 从服务器同步最新数据库
./scripts/docker-start.sh sync-db

# 重启服务
./scripts/docker-start.sh restart
```

### 7.4 端口被占用

```bash
# 查看端口占用
lsof -i :80
lsof -i :8088
lsof -i :3306

# 修改端口（编辑 docker/.env）
NGINX_HTTP_PORT=8080
APP_EXTERNAL_PORT=18088
MYSQL_PORT=13306
```

```

   ├─ 规格管理
   ├─ 关键词管理

   └─ OCR 识别

📁 平台设置
   ├─ 推送设置
   │   ├─ 推送组管理（测试/活跃/打捞/潜水）
   │   ├─ 定时推送
   │   └─ 推送记录
   ├─ 用户管理
   │   ├─ 用户列表
   │   ├─ 用户行为（收藏/足迹）
   │   └─ 用户反馈
   ├─ 广告管理
   │   ├─ 广告位
   │   ├─ 专题管理
   │   └─ 穿搭推荐
   ├─ 促销管理（合并为一页）
   │   ├─ 限时特价
   │   ├─ 优惠券
   │   └─ 满减（隐藏）
   ├─ 活动管理
   ├─ 店铺设置
   │   ├─ 门店信息
   │   └─ 导购管理
   └─ 系统配置

       ├─ 微信配置
```
